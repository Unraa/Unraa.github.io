# Projects

## IPhone Comparsion Website

### 프로젝트 소개
아이폰 17 시리즈 모델의 스펙을 비교할 수 있는 웹 페이지를 제작하였습니다.

### 개발 목적
사용자가 모델의 정보를 직관적으로 비교할 수 있도록 구현하는 것을 목표로 하였습니다.

### 사용 기술
- HTML
- CSS
- JavaScript

### 주요 기능
- 모델별 스펙 비교
- 반응형 UI 구현

### 맡은 역할
프론트엔드 UI 구성 및 데이터 비교 기능 구현

### 어려웠던 점
여러 제품 정보를 한 화면에서 가독성 있게 배치하는 과정에서 UI 구조 설계에 어려움이 있었습니다.

### 개선 방향
검색 기능 및 가격 비교, 필터 기능을 추가하고 여러 휴대폰 기종들을 추가하여 사용성을 개선할 예정입니다.

## AI Chatbot for Computer Science Students

### 프로젝트 소개
한국외국어대학교 컴퓨터공학과 학생들이 전공 관련 정보를 쉽게 확인할 수 있도록 AI 기반 챗봇 서비스를 개발하였습니다.

### 개발 목적
학과 정보, 전공 질문, 졸업 요건 질문 등에 빠르게 접근할 수 있는 서비스를 제공하기 위해 제작하였습니다.

### 사용 기술
- Python
- OpenAI API
- HTML
- CSS
- JavaScript

### 주요 기능
- 사용자 질문 응답
- 전공 관련 정보 제공
- 간단한 대화형 인터페이스 구현

### 맡은 역할
AI 프롬프트 및 가독성을 위한 챗봇 인터페이스 조정

### 어려웠던 점
질문의 의도를 정확히 파악하여 적절한 응답을 반환하도록 프롬프트를 구성하는 과정이 어려웠습니다.

### 개선 방향
추후 학과 공지사항 및 커뮤니티 기능 포함 예정
